### Overview 

This microservice handles authentication and authorization using OAuth2 and JWT. It provides 
secure token issuance, validation, and user authentication for client applications and downstream 
services. Built with Spring Security and Spring Authorization Server, it integrates seamlessly with 
other microservices in the ecosystem.
